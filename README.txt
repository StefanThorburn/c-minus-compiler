***** CHECKPOINT ONE *****
Our scanner uses the provided sample scanner as a base


*** RUNNING ***
In root directory, type 'make' or 'make scanner'
Run scanner in root directory with
    'java -cp bin/ Scanner < input_file'
Note that it won't work with windows powershell (doesn't allow < operator)

Type 'make clean' to remove compiled files
